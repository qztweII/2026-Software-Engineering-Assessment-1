# Fruit API Application

It is an app that searches and compares fruit and its nutritions in a fruits data API. 

This is my 2026 Software Engineering Assessment 1. 

You can:
- Search fruits by name
- Filter fruits by nutritional value
- Graphically compare fruits by nutrtitional value

## How to use:
1. In the terminal run: `pip install -r requirements.txt` to install all the dependencies

2. Run main.py
